# Seguridad

No publiques vulnerabilidades, rutas privadas, números de serie ni diagnósticos completos en una incidencia pública.

Si encuentras un problema de seguridad, comunícalo de forma privada mediante la opción **Security → Report a vulnerability** de GitHub o escribe a [randomlabdev@gmail.com](mailto:randomlabdev@gmail.com). Incluye la versión afectada, los pasos de reproducción y el impacto, evitando adjuntar datos personales innecesarios.

PcTemp necesita privilegios de administrador para consultar sensores de bajo nivel. Descarga instaladores únicamente desde las publicaciones oficiales del repositorio y verifica el SHA-256 publicado con cada versión.
